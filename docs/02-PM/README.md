## 📖 我的交付物

每个 PM 工作日志（`work-logs/pm-doc-vX.0.md`）包含：

1. **Background** - 为什么做这个功能
2. **User Story** - 作为[角色]，我希望[功能]，以便[目标]
3. **Acceptance Criteria** - 具体的验收标准（AC-*）
4. **Out of Scope** - 不在本次范围内
5. **Success Metrics** - 成功指标

完成后，累加到 [baseline.md](baseline.md)
