# 更新日志（用户向）

> 面向使用者、按"能感知的变化"提炼自内部版本与 TODO（非逐条工程记录）。最新在上。

## 2026-04 — 导入与可靠性
- **从 JSON 导入工作流**：粘贴/上传工作流 JSON，校验后一键替换画布；自动剥离 markdown 代码围栏，显示预览摘要。
- **导入校验更贴合实际**：接受全部 6 种节点类型（CONSUMER / CONSUMERWITHOUTERROR / IFELSE / MESSAGE / FUNCTION_V2 / FUNCTION_V3）；对 IFELSE 分支边、外部/字符串 ID 放宽校验，减少误报。
- **一键发布（Deploy）**：从 Hub 把应用与工作流发布到目标环境；跨域自动经代理处理。
- **表单/对比度打磨**：输入框与按钮高度统一；主/危险按钮文字对比度达 WCAG AA。

## 2026-04 — 全新 IBM Carbon 界面
- **前端按设计稿重建**：IBM Carbon 设计语言，IBM Plex 字体、0px 直角、统一主色，桌面与移动端响应式。
- **节点编辑器升级**：可调整宽度、默认只读 + 编辑按钮、关闭按钮修复、无障碍（aria-label、对比度合规）。
- **节点调色板**：每种节点带单行说明；复制/删除节点时自动清理连线。
- **规则键校验**：规则必须是单个有效 JSONPath，失焦即校验、错误时禁用保存。

## 2026-04 — AI 辅助
- **Explain**：用自然语言解释一个工作流在做什么。
- **Generate**：用一句话生成工作流骨架；生成结果在应用前做结构校验。

## 既有能力（基线）
- 应用列表/搜索/分页、创建/删除、设置编辑、跨应用复制、配置历史与回滚。
- 画布编排（节点/连线/配置/保存/试跑）。
- 在线执行：丰富化→下发两阶段、同步/异步、幂等、应用级重试。
- 运行记录：筛选、下钻、父子/重试链。
